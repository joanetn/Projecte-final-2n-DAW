================================================================================
PRÓXIMOS PASOS PARA EJECUTAR EL GATEWAY
================================================================================

1. Abrir terminal en la carpeta del proyecto

2. Ejecutar:
   npm run gateway

3. Esperar a que se inicien los 7 servidores. Deberías ver algo como:

   [gateway] ► Listening on http://localhost:8000 (GATEWAY_ENABLED=true)
   [users]   ► Listening on http://localhost:8001
   [clubs]   ► Listening on http://localhost:8002
   [leagues] ► Listening on http://localhost:8003
   [matches] ► Listening on http://localhost:8004
   [venues]  ► Listening on http://localhost:8005
   [lineup]  ► Listening on http://localhost:8006

4. Ahora puedes probar el gateway desde otra terminal:

   # Petición a través del gateway (puerto 8000)
   curl http://localhost:8000/api/clubs
   
   # La petición se enruta automáticamente a puerto 8002 (Club Service)

================================================================================
PROBLEMAS SI OCURREN
================================================================================

Si aún no funciona, revisa:

1. ¿Están los puertos 8000-8006 disponibles?
   Windows: netstat -ano | findstr :8000
   Linux/Mac: lsof -i :8000

2. ¿Está instalado cross-env?
   npm install cross-env concurrently

3. ¿Está Apache/otro servidor usando los puertos?
   Detener Apache u otros servidores

4. ¿Errores de PHP?
   cd backend_laravel
   php artisan config:cache
   php artisan route:cache
   php artisan optimize:clear

5. Ver logs de un servicio específico:
   cd backend_laravel
   cross-env GATEWAY_ENABLED=false php artisan serve --port=8006
